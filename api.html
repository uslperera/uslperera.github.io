co_occurrence_search(query, qtm)¶

Search for a document using co-occurrence of words.

main.ss_similarity(documents, new_document, hal_model, count)¶

Find documents using SemSimilar similarity.

Both HAL and Lesk based similarity calculations are used to find the most related documents.

Parameters:
  • documents (list<semsimilar.core.model.document.Document>) – documents list
  • new_document (semsimilar.core.model.document.Document) – document to search
  • hal_model (semsimilar.core.similarity.corpus.hal.Hal) – HAL model created from existing documents
  • count (int) – number of results wanted
Returns:

Top matched documents with their scores (0-1)
Return type:

list<(semsimilar.core.model.document.Document, float)>
Example: 
>>> doc = Document(101, "PHP Session Security", 
    "What are some guidelines for maintaining 
    responsible session security with PHP", 
    "<security><php>")
>>> ss_similarity(documents, doc, hal, 1)
[(document, 0.708)]
