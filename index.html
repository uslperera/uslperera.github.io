SemSimilar¶

Welcome to SemSimilar’s documentation!¶

SemSimilar is a Python library for information retrieval, document indexing and similarity retrieval with large corpora. Target audience is the natural language processing (NLP) and information retrieval (IR) community. It uses corpus-based and knowledge-based techniques to find semantically similar documents.

Features¶

  • Simple & Fast.
  • Boilerplate data structure.
  • Algorithms from Distributional Semantics, Hyperspace Analogue to Language (HAL).
  • No prior knowledge needed of the corpora.
  • Integrated with WordNet which enables to find more relations such as hypernym/hyponym.
  • Multicore processing of large corpora.
